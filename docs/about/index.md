# About Library Skills

## Agent Skills

**Library Skills** uses the same [Agent Skills](https://agentskills.io/) standard, with the additional definition of how to bundle skills inside of library packages, but it's the same Agent Skills standard.

Nevertheless, it is not associated with [Agent Skills](https://agentskills.io/), it's an independent project.

## Alternatives

There are other tools and ideas about sharing and distributing skills, but in most cases they refer to generic skills, not specific library skills, bundled and in sync with each version.
