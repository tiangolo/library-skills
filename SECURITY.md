# Security Policy

Security is very important for Library Skills and its community. 🔒

Learn more about it below. 👇

## Versions

The latest versions of Library Skills are supported.

## Reporting a Vulnerability

If you think you found a vulnerability, and even if you are not sure about it, please [report it](https://github.com/tiangolo/library-skills/security).

I (the author, [@tiangolo](https://twitter.com/tiangolo)) will review it thoroughly and get back to you.

## Public Discussions

Please restrain from publicly discussing a potential security vulnerability. 🙊

It's better to discuss privately and try to find a solution first, to limit the potential impact as much as possible.

Thanks for your help!
